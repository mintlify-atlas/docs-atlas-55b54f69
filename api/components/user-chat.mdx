## Overview

The `UserChat` component provides a conversational interface for users to interact with the LLM. It displays a chat history with user queries and AI responses, and includes a text input area for submitting new queries. When a query is submitted, it triggers the API call to fetch data including LLM responses, stock dataframes, and news articles.

## Import

```jsx
import UserChat from "./Components/UserChat";
```

## Internal State

The component manages its own internal state:

- `isReceived` (boolean): Tracks whether a response has been received. Controls the send button icon (SendIcon vs CircularProgress).
- `responseHistory` (array): Stores all previous query-response pairs as tuples `[userQuery, aiResponse]`.

## Usage Example

```jsx
import { useState } from "react";
import UserChat from "./Components/UserChat";
import { client } from "@gradio/client";

async function fetchResponse(
  userQuery,
  setDisplay,
  setIsReceived,
  setArticles,
  setDataframe,
) {
  const app = await client("http://127.0.0.1:7860/");
  const result = await app.predict("/predict", [[userQuery]]);
  setIsReceived(true);
  setDisplay(result.data[0]);
  setDataframe(result.data[1]);
  setArticles(result.data[2]);
}

function App() {
  const [displayedResponse, setDisplay] = useState("");
  const [userQuery, setUserQuery] = useState("");
  const [articles, setArticles] = useState("");
  const [dataframe, setDataframe] = useState({});

  return (
    <UserChat
      displayedResponse={displayedResponse}
      setDisplay={setDisplay}
      userQuery={userQuery}
      setUserQuery={setUserQuery}
      fetchResponse={fetchResponse}
      setArticles={setArticles}
      setDataframe={setDataframe}
    />
  );
}
```

## Component Behavior

### Chat History Display

The component maintains a conversation history in `responseHistory`. Each entry is rendered as:
- User query: displayed in a teal bubble (`bg-teal-900`) aligned to the start
- AI response: displayed in a slate bubble (`bg-slate-800`) aligned to the end

### Query Submission

When the send button is clicked (`UserChat.jsx:72-84`):

1. Adds a temporary entry to `responseHistory` with the user query
2. Toggles `isReceived` to `false` to show loading indicator
3. Calls `fetchResponse` with the query and all state setters
4. The send button icon changes from `<SendIcon />` to `<CircularProgress />` while loading

### Response Updates

The `useEffect` hook on line 27 watches for changes to `displayedResponse` and appends new responses to the history array.
