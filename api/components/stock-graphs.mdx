## Overview

The Bounce Project includes two components for rendering stock market visualizations:
- **StockHistoryGraph**: Displays time-series stock price data using Plotly
- **StockHistogram**: Renders sample bounce rate data using CanvasJS

---

## StockHistoryGraph

Renders a line chart showing stock price history over time using Plotly.js.

### Import

```jsx
import StockHistoryGraph from "./Components/StockHistoryGraph";
```

### Props

<ParamField path="dataframe" type="object">
  The stock data object returned from the API. Expected structure:
  ```typescript
  {
    data: Array<[string, ...any, number]> // Array of rows
    // data[i][0] = date string
    // data[i][18] = price value
  }
  ```
  When `null` or undefined, displays a placeholder message.
</ParamField>

### Data Processing

The component extracts:
- **X-axis (dates)**: Maps `dataframe.data` array index 0 from each row
- **Y-axis (prices)**: Maps `dataframe.data` array index 18 from each row

### Plotly Configuration

The graph is rendered with the following options:

```javascript
const plotData = [
  {
    x: dx,              // Date values
    y: dy,              // Price values
    type: "scatter",
    mode: "lines",
    marker: { color: "white" },
  },
];

const layout = {
  title: "Stock History",
  xaxis: { title: "Date" },
  yaxis: { title: "Price" },
  plot_bgcolor: "black",
  paper_bgcolor: "black",
  font: { color: "white" },
};
```

The chart is rendered into a DOM element with ID `stock-graph`.

### Usage Example

```jsx
import { useState } from "react";
import StockHistoryGraph from "./Components/StockHistoryGraph";

function App() {
  const [dataframe, setDataframe] = useState(null);

  // dataframe gets populated by the API response
  // result.data[1] contains the stock data

  return (
    <div>
      <StockHistoryGraph dataframe={dataframe} />
    </div>
  );
}
```

### Conditional Rendering

The component checks if data exists before rendering:
- If `dataframe` is `null` or `dataframe.data` is undefined: shows placeholder text "Ask a query to generate results"
- If data exists: renders the Plotly chart with stock history

## StockHistogram

Displays a line chart visualization for bounce rate data using CanvasJS. Currently uses static sample data.

### Import

```jsx
import StockHistogram from "./Components/StockHistogram";
```

### Props

<ParamField path="listOfPrices" type="array">
  Intended to receive price data for the histogram. Currently not used in the implementation - the component renders static sample data instead.
</ParamField>

### Chart Configuration

The component renders a CanvasJS line chart with the following options:

```javascript
const options = {
  animationEnabled: true,
  exportEnabled: true,
  theme: "dark2",
  title: {
    text: "Bounce Rate by Week of Year",
  },
  axisY: {
    title: "Bounce Rate",
    suffix: "%",
  },
  axisX: {
    title: "Week of Year",
    prefix: "W",
    interval: 2,
  },
  data: [
    {
      type: "line",
      toolTipContent: "Week {x}: {y}%",
      dataPoints: [
        { x: 1, y: 64 },
        { x: 2, y: 61 },
        // ... 23 weeks of static data
      ],
    },
  ],
};
```

### Usage Example

```jsx
import StockHistogram from "./Components/StockHistogram";

function App() {
  const prices = []; // Currently not consumed by the component

  return (
    <div>
      <StockHistogram listOfPrices={prices} />
    </div>
  );
}
```

### Static Data Points

The component currently renders 23 weeks of hardcoded bounce rate data (weeks 1-23) with percentage values ranging from 53% to 64%.

### Dependencies

- **CanvasJS**: `@canvasjs/react-charts`
- **React**: Core library

### Styling

- Container: `flex h-full w-full justify-center items-center m-4 relative bottom-24`
- Uses CanvasJS "dark2" theme for dark mode appearance

### Note

The `listOfPrices` prop is accepted but not currently utilized in the component implementation. The chart always displays the static sample data defined in the options object.
