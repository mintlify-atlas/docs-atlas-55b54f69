## Stock History Graph

The primary visualization is an interactive stock price chart powered by **Plotly.js**.

### What the Graph Shows

<Steps>
  <Step title="Historical Data">
    Past stock prices from Yahoo Finance, starting from 2020-01-01 up to the query date.
  </Step>
  
  <Step title="Predicted Prices">
    365-day forecast generated by Facebook Prophet model showing potential future prices.
  </Step>
  
  <Step title="Trend Line">
    Continuous line connecting historical and predicted data for easy trend identification.
  </Step>
</Steps>

```jsx Graph Implementation (StockHistoryGraph.jsx)
const StockHistoryGraph = ({ dataframe }) => {
  useEffect(() => {
    if (dataframe !== null && dataframe.data) {
      // Extract date (x-axis) and price (y-axis) data
      const dx = dataframe.data.map((item) => item[0]);  // Dates
      const dy = dataframe.data.map((item) => item[18]); // Predicted prices

      const plotData = [{
        x: dx,
        y: dy,
        type: "scatter",
        mode: "lines",
        marker: { color: "white" }
      }];

      const layout = {
        title: "Stock History",
        xaxis: { title: "Date" },
        yaxis: { title: "Price" },
        plot_bgcolor: "black",
        paper_bgcolor: "black",
        font: { color: "white" }
      };

      Plotly.newPlot("stock-graph", plotData, layout);
    }
  }, [dataframe]);
};
```

<Info>
  The graph uses index `[18]` from the forecast dataframe, which corresponds to the `yhat` (predicted value) column from Facebook Prophet's output.
</Info>

### Reading the Graph

<Tabs>
  <Tab title="X-Axis (Dates)">
    **Time Axis**: Displays dates in chronological order
    
    - **Historical Period**: 2020-01-01 to query date (typically 2024-03-01)
    - **Forecast Period**: Next 365 days from the last historical date
    - **Total Range**: ~4 years of historical data + 1 year forecast
    
    <Note>
      The date range is currently hardcoded in the backend. Historical data starts from 2020-01-01.
    </Note>
  </Tab>
  
  <Tab title="Y-Axis (Price)">
    **Price Axis**: Shows stock price in the currency of the stock's trading market
    
    - **Auto-scaled**: Plotly automatically adjusts the scale based on price range
    - **Units**: Typically USD for US stocks
    - **Precision**: Displays prices to 2 decimal places
    
    <Tip>
      Hover over any point on the graph to see the exact price and date.
    </Tip>
  </Tab>
  
  <Tab title="Trend Line">
    **Continuous Line**: Connects all data points showing price movement
    
    - **Color**: White line on black background for high contrast
    - **Smoothness**: Line follows actual data points without interpolation
    - **Breaks**: No visual distinction between historical and predicted data
    
    <Warning>
      The graph doesn't visually separate historical from predicted data. Use the date on the x-axis to distinguish between actual and forecasted prices.
    </Warning>
  </Tab>
</Tabs>

## Understanding the Forecast Data

The price predictions are generated using **Facebook Prophet**, a time-series forecasting model.

### How Forecasts Are Created

```python Forecast Generation Process (app.py)
def fetch_stock_data(ticker_symbol, start_date, end_date):
    # Download historical data from Yahoo Finance
    stock_data = yf.download(ticker_symbol, start=start_date, end=end_date)
    df = stock_data[['Adj Close']].reset_index()
    
    # Rename columns for Prophet (requires 'ds' and 'y')
    df = df.rename(columns={'Date': 'ds', 'Adj Close': 'y'})
    return df

def train_prophet_model(df):
    # Train Prophet model on historical data
    model = Prophet()
    model.fit(df)
    return model

def make_forecast(model, periods):
    # Generate 365-day forecast
    future = model.make_future_dataframe(periods=periods)
    forecast = model.predict(future)
    return forecast

# Usage
ticker_symbol = "GOOGL"
start_date = "2020-01-01"
end_date = "2024-03-01"
df = fetch_stock_data(ticker_symbol, start_date, end_date)
model = train_prophet_model(df)
forecast = make_forecast(model, 365)  # 365-day prediction
```

<Steps>
  <Step title="Data Collection">
    Historical adjusted close prices are fetched from Yahoo Finance.
  </Step>
  
  <Step title="Model Training">
    Prophet model learns seasonal patterns, trends, and cycles from historical data.
  </Step>
  
  <Step title="Prediction">
    Model generates predictions for the next 365 days based on learned patterns.
  </Step>
</Steps>

<Info>
  **Adjusted Close Price**: The graph uses adjusted close prices, which account for stock splits, dividends, and other corporate actions. This provides a more accurate picture of actual returns.
</Info>

## Prophet Forecast Components

The forecast dataframe includes multiple columns:

| Column | Description | Used in Graph |
|--------|-------------|---------------|
| `ds` | Date/timestamp | Yes (x-axis) |
| `yhat` | Predicted value | Yes (y-axis) |
| `yhat_lower` | Lower confidence bound | No |
| `yhat_upper` | Upper confidence bound | No |
| `trend` | Overall trend component | No |
| `seasonal` | Seasonal patterns | No |

<Note>
  Currently, only the main prediction (`yhat` at index 18) is visualized. Confidence intervals are calculated but not displayed on the graph.
</Note>

## Technical Implementation Details

### Dataframe Structure

The forecast data arrives in this format:

```javascript Dataframe Processing
// Raw dataframe from backend
const dataframe = {
  data: [
    ["2020-01-01", val1, val2, ..., val18, ...],  // Historical point
    ["2020-01-02", val1, val2, ..., val18, ...],
    // ... more data points ...
    ["2025-03-01", val1, val2, ..., val18, ...]   // Forecast point
  ]
};

// Extraction for plotting
const dx = dataframe.data.map((item) => item[0]);   // Dates (x-axis)
const dy = dataframe.data.map((item) => item[18]);  // Prices (y-axis)
```

<Info>
  Each data point is an array where index 0 is the date and index 18 is the predicted price (`yhat` from Prophet).
</Info>

### Plotly Configuration

The graph uses Plotly's scatter plot with line mode:

```javascript Plotly Configuration
const plotData = [{
  x: dx,                    // Date array
  y: dy,                    // Price array
  type: "scatter",          // Scatter plot type
  mode: "lines",            // Line connection mode
  marker: { color: "white" } // Line color
}];

Plotly.newPlot("stock-graph", plotData, layout);
```

<Note>
  The graph uses `mode: "lines"` to connect all points with a continuous line. This creates a smooth visualization of price changes over time.
</Note>

## Limitations of Visualizations

<AccordionGroup>
  <Accordion title="No confidence intervals displayed">
    Prophet generates upper and lower bounds for predictions, but these aren't currently shown on the graph. The actual price could be higher or lower than the displayed prediction.
  </Accordion>
  
  <Accordion title="Fixed historical date range">
    The start date (2020-01-01) is hardcoded. You can't adjust the historical period to view longer or shorter timeframes.
  </Accordion>
  
  <Accordion title="No volume data">
    Trading volume isn't displayed, which is an important indicator of market strength.
  </Accordion>
  
  <Accordion title="Single stock only">
    You can't compare multiple stocks on the same graph or view market indices for context.
  </Accordion>
  
  <Accordion title="No technical indicators">
    Common technical analysis tools (moving averages, RSI, MACD, etc.) aren't included.
  </Accordion>
</AccordionGroup>

<Warning>
  Use external charting tools for advanced technical analysis. The Bounce Project's visualizations are designed for quick trend assessment, not comprehensive technical analysis.
</Warning>

## Troubleshooting Visualizations

<AccordionGroup>
  <Accordion title="Graph not appearing">
    Ensure you've submitted a query and the backend has returned forecast data. Check browser console for JavaScript errors.
  </Accordion>
  
  <Accordion title="Graph shows flat line">
    This may indicate insufficient historical data or a data processing error. Try querying a different, more established stock.
  </Accordion>
  
  <Accordion title="Dates not displaying correctly">
    Check that your browser timezone is set correctly. Plotly uses local timezone for date rendering.
  </Accordion>
  
  <Accordion title="Interactive features not working">
    Ensure JavaScript is enabled and you're using a modern browser. Plotly requires JavaScript for interactivity.
  </Accordion>
</AccordionGroup>
