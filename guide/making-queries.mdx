The Bounce Project uses advanced natural language processing to understand your stock-related questions and provide intelligent recommendations.

## How Queries Work

When you submit a query, The Bounce Project:

1. **Extracts Stock Information**: Identifies which stock you're asking about using AI text generation
2. **Fetches Real-time News**: Gathers recent headlines and articles about the stock
3. **Analyzes Sentiment**: Performs sentiment analysis on news articles using a RoBERTa-based model
4. **Generates Predictions**: Creates price forecasts using Facebook Prophet model
5. **Provides Recommendations**: Delivers actionable insights based on combined analysis

<Info>
  The system uses the `cardiffnlp/twitter-roberta-base-sentiment-latest` model for sentiment analysis, trained specifically for understanding financial market sentiment.
</Info>

## Query Processing Flow

Here's what happens when you submit a query:

```javascript Client-Side Processing (App.jsx)
async function fetchResponse(
  userQuery,
  setDisplay,
  setIsReceived,
  setArticles,
  setDataframe,
) {
  try {
    console.log("You entered", userQuery);
    
    // Connect to Gradio backend
    const app = await client("http://127.0.0.1:7860/");
    
    // Send prediction request
    const result = await app.predict("/predict", [[`${userQuery}`]]);
    
    // Process results:
    setDisplay(result.data[0]);      // AI recommendation text
    setDataframe(result.data[1]);    // Stock price forecast data
    setArticles(result.data[2]);     // News headlines
    
    setIsReceived(true);
  } catch (error) {
    console.log("Error handled: ", error);
  }
}
```

## Technical Implementation

For developers interested in the backend processing:

```python Main Query Processing (app.py)
def main(prompt):
    # Extract stock information
    name, ticker = fetch_stock_name(prompt)
    
    # Analyze news sentiment
    neg, neu, pos, news = fetch_news_sentiment(name)
    sum = neg + neu + pos
    neg = neg / sum
    neu = neu / sum
    pos = pos / sum
    
    # Generate recommendation based on sentiment
    if neu > 0.55:
        # Stock is stable - hold recommendation
    elif pos > neg:
        # Stock has momentum - buy recommendation
    else:
        # Stock declining - sell recommendation
    
    # Generate forecast
    graph = make_graph(ticker)
    
    return result, graph, news
```

<Tip>
  The sentiment threshold of 0.55 for neutral sentiment is calibrated to avoid false positives. Adjust your expectations based on current market volatility.
</Tip>

## Troubleshooting Queries

<AccordionGroup>
  <Accordion title="No results returned">
    Ensure the stock ticker exists on Yahoo Finance and the company name is spelled correctly.
  </Accordion>
  
  <Accordion title="Incorrect stock identified">
    Try using the exact ticker symbol (e.g., "AAPL") instead of the company name if there's ambiguity.
  </Accordion>
  
  <Accordion title="Loading takes too long">
    The system fetches real-time news and processes sentiment analysis, which can take 10-30 seconds. Be patient while the AI analyzes the data.
  </Accordion>
  
  <Accordion title="Connection error">
    Ensure the Gradio backend is running on http://127.0.0.1:7860/ and your network connection is stable.
  </Accordion>
</AccordionGroup>
