## How It Works

<Steps>
  <Step title="Ask in Natural Language">
    Simply type your question about any stock in plain English, like "Should I invest in Apple?"
  </Step>
  <Step title="AI Analyzes Market Data">
    The system extracts the stock ticker, fetches news headlines, and analyzes sentiment using AI models
  </Step>
  <Step title="Get Predictions & Recommendations">
    View price forecasts, sentiment analysis, and conversational recommendations to guide your decision
  </Step>
  <Step title="Visualize Insights">

## Technology Stack

Built with modern technologies for performance and scalability:

- **Frontend**: React + Vite, Material UI, NextUI, Plotly.js
- **Backend**: Python with Gradio API server
- **AI/ML**: Transformers (sentiment analysis), Prophet (forecasting)
- **Data**: yfinance for stock data, web scraping for news

<Warning>
  The Bounce Project provides AI-powered recommendations based on sentiment analysis and historical data. Always conduct your own research and consult with financial advisors before making investment decisions.
</Warning>
